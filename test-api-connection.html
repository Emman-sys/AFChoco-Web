Troubleshooting Steps

  1. Check if Node server is running: lsof -i :3000
  2. Verify the API works: curl http://localhost:3000/api/health
  3. Check browser console (F12) for CORS errors
  4. Ensure you're accessing via http://localhost:8000
  5. Try disabling browser extensions that might block requests
  6. Check if your firewall is blocking localhost connections
